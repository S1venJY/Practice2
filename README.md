# RemoteGalleryService

Вебсервіс віддаленого доступу до персоналізованої галереї зображень. Створено в рамках навчальної практики.

## Стек технологій
* **Backend:** C# (.NET 8.0), ASP.NET Core Web API
* **Frontend:** HTML5, CSS3 (Grid Layout), JavaScript (Fetch API)
* **Безпека:** Хешування паролів за алгоритмом SHA-256
* **Збереження даних:** Локальні файли JSON (метадані) та фізичний диск (зображення)

Архітектура та структура коду
Проєкт реалізовано за принципом розділення відповідальності (Separation of Concerns).

Структура папок:
wwwroot/ — статичні файли фронтенду (index.html) та папка uploads для збережених фото.

Data/ — папка з JSON-файлами бази даних.

Program.cs — єдиний файл бекенду, що містить конфігурацію, моделі, сервіс та контролер.

Основні компоненти (в Program.cs):
Моделі даних: User (профіль), ImageRecord (метадані фото), Album (альбоми).

GalleryStorageService: Сервіс бізнес-логики. Відповідає за авторизацію, хешування паролів, асинхронне збереження файлів на диск із генерацією Guid та роботу з JSON-файлами в папці Data.

GalleryController: REST Web API контролер. Обробляє HTTP-запити від клієнта:

POST /auth/register та /auth/login — реєстрація та вхід.

POST /upload — завантаження фотографій.

POST /albums/create — створення альбомів.

GET /view/grid — отримання списку фото та альбомів користувача.

Клієнтська частина (index.html):
Працює асинхронно через Fetch API без перезавантаження сторінки.

Інтерфейс адаптивний, побудований на CSS Grid.

Реалізовано модальне вікно для перегляду картинок у повному розмірі при натисканні.
